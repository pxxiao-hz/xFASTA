# xFASTA
```shell
usage: xFASTA [-h]  ...

pxx 的分析小工具

positional arguments:

    fasta     处理 FASTA 文件
    stat      统计基因组的相关信息
    gff       Perform operations on GFF file
    gap       Genome gap statistics
    read      Reads information

optional arguments:
  -h, --help  show this help message and exit
```
